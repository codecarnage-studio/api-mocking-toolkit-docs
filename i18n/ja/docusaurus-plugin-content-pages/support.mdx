# サポート

**API Mocking Toolkit** を使用中に問題が発生した場合は、以下の方法でご連絡ください：

- メール：[support@codecarnage.com](mailto:support@codecarnage.com)
- GitHub Issues：https://github.com/codecarnage-studio/api-mocking-toolkit-docs/issues

Unity Asset Store の購入または請求に関するご質問は、Asset Store の請求情報を添えてお問い合わせください。
